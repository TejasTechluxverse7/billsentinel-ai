
# 🧾 BILLSENTINEL AI

**AI-powered system to detect errors in medical bills with clear explanations.**

BILLSENTINEL analyzes medical bills to find hidden mistakes such as duplicate charges, impossible timelines, and mismatched treatments — helping patients save money and understand their bills.

---

## 🚀 Features
- Detects post-discharge charges
- Identifies mismatched diagnosis and treatments
- Finds duplicate or hidden fees across pages
- Explains what is wrong and how to fix it
- Fast analysis (seconds per bill)
- Fully private processing

---

## 🧠 Architecture
- **Document Parser** – Extracts structured text from bills  
- **Timeline Analyzer** – Detects impossible billing sequences  
- **Logic Validator** – Checks medical consistency  
- **Explainer Model** – Generates human-readable explanations  

---

## 🛠️ Tech Stack
- Python
- PyTorch / TensorFlow
- Transformers
- OpenCV / OCR
- Streamlit (optional UI)
